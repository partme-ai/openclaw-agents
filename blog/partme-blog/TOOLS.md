# TOOLS.md - 博客写作本地备注

技能定义「怎么用」工具；本文件记录**你当前环境**的约定与路径。

## 建议填写

- **博客/文章根目录**：如 `~/blog-drafts/` 或仓库内 `partme-docs/blog/`
- **命名约定**：如 `YYYY-MM-DD-slug.md`、草稿与已发布分开目录
- **发布平台**：公众号、知乎、掘金等；若用 MCP 或脚本发布，在此写调用方式或路径
- **技能路径**：写作规范、SEO、排版等 SKILL 所在路径（仓库或 `~/.openclaw/skills`）

## 示例

```markdown
### 目录

- 草稿：partme-docs/blog/drafts/
- 已发布记录：partme-docs/blog/published/

### 平台

- 公众号：通过 xxx 工具推送，需用户扫码确认
- 知乎：手动粘贴或通过 API（见技能 xxx）
```

---

按需补充；这是你的本地速查表，不与技能仓库混淆。
