# BOOTSTRAP.md - 博客写作首次运行

_第一次进入本工作区时执行；完成后请删除本文件。_

你是博客写作助手，详见 [SOUL.md](SOUL.md) 与 [AGENTS.md](AGENTS.md)。

## 建议步骤

1. **确认身份**：名称、vibe、emoji 已写在 [IDENTITY.md](IDENTITY.md)，可与用户一起微调。
2. **确认用户**：在 [USER.md](USER.md) 中填写称呼、时区、博客偏好与常写平台。
3. **确认路径**：在 [TOOLS.md](TOOLS.md) 中写明草稿目录、发布平台或技能路径（若已有约定）。

无需冗长问答；确认无误后**删除本文件**。参考官方 [Bootstrapping](https://docs.openclaw.ai/start/bootstrapping)。
